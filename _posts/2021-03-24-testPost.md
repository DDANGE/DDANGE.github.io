깃허브 페이지 시작하기
=============
 <!-- 큰제목 밑에 =====헤더 타이틀 -->
이전 회사에서 협업툴로 SVN을 사용해 보았지만, 깃허브를 많이 사용했다.
포트폴리오 관리나 주기적으로 공부하기 위해서 필수였다.
 
깃허브를 공부하고 잘 사용하지 않아 공부했던 내용이 잊혀지기 일수였다.
그래서 블로그를 깃허브 페이지로 옮기고 깃에 업데이트 하기로 했다
 
 오늘은 첫번째 포스트일텐데 여러가지 마크다운 문법을 적용해보고 소스트리를 통해 업로드 할 생각이다.

 c#언어 코드 테스트
 ```c#
  Console.WriteLine("C# 코드 테스트");
  public class Test
  {
  }
 ```
c언어 코드 테스트
 ```c
  printf("C언어 코드 테스트");
 ```
